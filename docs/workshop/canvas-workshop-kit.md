# Shared Canvas Workshop Kit (Facilitator Draft)

This kit packages everything needed to run the cross-functional workshop and give participants both textual and visual material.

## 1. Session Overview

- **Audience**: Product, Design, Engineering, Strategy, Research
- **Duration**: 120 minutes (hybrid-friendly)
- **Goal**: Align on initial shared canvas schema, confirm core workflows, and capture action items for building the integrated workspace.

- Use bold color coding: **Story (Purple)**, **Strategy (Teal)**, **Execution (Amber)**, **Intelligence (Blue)** to match entity groups.

- Provide Slide Notes referencing file paths for deeper reading.
