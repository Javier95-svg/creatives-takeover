# Shared Canvas Data Model (Draft)

This draft maps the foundation for a cross-tool canvas that unifies BizMap AI conversations, Prompt Library assets, and Insighta signals. It mixes structural notes with lightweight visualizations to accelerate discussion during the workshop.

## Core Entities

| Entity | Purpose | Key Fields | Source / Owner |
|--------|---------|------------|----------------|
| `WorkspaceCanvas` | Parent record for an ideation/planning workspace | `id`, `title`, `founder_profile`, `stage`, `goal`, `created_at`, `updated_at` | Product |
| `IdeaBrief` | Structured summary generated from BizMap AI discovery dialogs | `canvas_id`, `problem`, `audience`, `solution_snapshot`, `evidence`, `confidence_score`, `last_updated_by` | BizMap AI |
| `PromptRun` | Log of Prompt Library sequences executed (manual or via agent) | `canvas_id`, `prompt_pack_id`, `entry_point`, `outputs`, `status`, `owner` | Content Ops |
| `PlanSprint` | 2-4 week execution sprint container | `canvas_id`, `sprint_number`, `focus_theme`, `start_date`, `end_date`, `status` | Strategy |
| `Task` | Action items tied to sprints or agent suggestions | `sprint_id`, `source` (agent, user, insighta), `description`, `priority`, `assignee`, `due_date`, `status`, `evidence_link` | Execution |
| `InsightEvent` | Captured Insighta signal (funding, trend, metric) | `canvas_id`, `type`, `title`, `summary`, `data_source`, `link`, `impact_score`, `follow_up_task_id` | Intelligence |
| `FeedbackSignal` | Check-ins and sentiment from chatbot or teammates | `canvas_id`, `origin` (BizMap AI micro-check, founder note), `score`, `notes`, `timestamp` | Research |
| `Artifact` | Exported documents (pitch deck, roadmap PDF, experiment log) | `canvas_id`, `category`, `url`, `version`, `created_by`, `visibility` | Ops |

### Lifecycle Notes

1. **Canvas Creation**  
   - Triggered when a founder starts a new BizMap AI journey or imports an existing project. Seed with persona, stage, and initial goals.
2. **Discovery → IdeaBrief**  
   - BizMap AI parses early conversation states into `IdeaBrief` sections; outputs are editable by humans. Track version history to reflect refinement.
3. **Prompt Selection**  
   - Prompt Library cards selected manually or by the agent create `PromptRun` entries. Results flow into IdeaBrief or PlanSprint depending on template type.
4. **Planning Sprints**  
   - `PlanSprint` holds sprint theme and timeline. Tasks generated by the agent or team attach to the sprint; tasks may reference prompts or insight events.
5. **Insight Feedback Loop**  
   - Insighta events attach to the canvas and can spawn tasks. When a task resolves, capture outcome values back on the originating `InsightEvent`.
6. **Feedback & Sentiment**  
   - Micro check-ins, survey responses, and team comments log as `FeedbackSignal` to inform prioritization and agent tone adjustments.
7. **Artifacts & Sharing**  
   - Exported deliverables attach as `Artifact` records; connect to relevant tasks or sprints for traceability.
